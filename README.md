## OPEN API

고속도로 공공데이터 포털에서 제공하는 OPEN API 이용

http://data.ex.co.kr/openapi/intro/introduce02

## 기능 사항

1. 전국 휴게소 리스트
   - 전국 휴게소 위치 제공
   - 현재 있는 위치에서 가장 가까운 휴게소 리스트 제공

2. 휴게소에서 제공하는 음식 리스트
   - 추천, 베스트, 프리미엄 음식 필터링
   - 가격순 소팅
